📋 Python To-Do List (CLI)

A simple command-line To-Do List application built with Python.
This project demonstrates Object-Oriented Programming (OOP) concepts and basic task management functionality.

---

🚀 Features

- Add new tasks
- View all tasks
- Delete tasks
- Menu-based interaction
- Built using Python classes and methods

---

🛠 Technologies Used

- Python 3
- Object-Oriented Programming (OOP)

---

▶️ How to Run

1. Download or clone the repository
2. Open terminal in the project folder
3. Run the program

python todo.py
